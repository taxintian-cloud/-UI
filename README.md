# 白紙から作ったタブ＆アコーディオンUI
URLはこちら⇒https://taxintian-cloud.github.io/-UI/


## 概要
HTML構造を見ずに、JavaScriptだけでタブUIとアコーディオンを作成しました。
さらに関数化して、複数セクションでも独立して動くようにしています。

- タブはクリックで切り替え可能
- アコーディオンはクリックで開閉可能
- 関数化により、どのセクションにも簡単に適用可能

## 開発で苦労した点
- 初めは **HTMLを見ながらしか書けなかった** JSを、完全白紙で書く練習を繰り返しました
- 複数セクションに対応させるために、`root` を意識して **関数化** することに苦戦
- アコーディオンとタブが **干渉せず独立して動くようにする** のも意外と難しかったです

## 学んだこと
- `root.querySelectorAll` を使うことで、**セクションごとに安全に要素を取得**できる
- 関数化して汎用性を持たせると、**複数箇所でも同じUIが使える**ことがわかった
- 白紙から書くことで、**JSのクリックイベントやクラス切替の流れを脳に刻み込める**

## スクリーンショット
- タブ切替後の画面
- アコーディオン開閉後の画面
- 複数セクションが独立して動作している様子
